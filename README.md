A full-stack ML-powered workout recommendation system.

## Tech Stack
- Node.js + Express + EJS
- Python + Flask
- Scikit-learn (Random Forest)

## How it works
1. User enters fitness details
2. Node.js sends data to ML API
3. ML predicts workout split
4. Backend renders full workout plan
